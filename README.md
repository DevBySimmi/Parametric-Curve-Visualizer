# Parametric-Curve-Visualizer
A Python-based visualization project that demonstrates various parametric curves including Circle, Ellipse, Lissajous Curve, Cycloid, Rose Curve, and Archimedean Spiral using NumPy and Matplotlib. The project provides an elegant graphical representation of mathematical curves with customizable parameters and enhanced visualization features.

## Features

- Circle Visualization
- Ellipse Visualization
- Lissajous Curves
- Cycloid Curves
- Rose Curves
- Archimedean Spirals
- Dark Theme Support
- Grid and Axis Visualization
- High-Resolution Plotting
- Clean Multi-Panel Layout

## Curves Included

### Circle
x = cos(t)

y = sin(t)

### Ellipse
x = a cos(t)

y = b sin(t)

### Lissajous Curve
x = sin(3t)

y = sin(2t)

### Cycloid
x = r(t − sin(t))

y = r(1 − cos(t))

### Rose Curve
x = cos(kt)cos(t)

y = cos(kt)sin(t)

### Archimedean Spiral
x = r cos(t)

y = r sin(t)

where r = t
